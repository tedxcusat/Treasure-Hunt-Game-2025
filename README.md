# GeoQuest: AR Treasure Hunt (v2)

Augmented Reality Location-Based Game built for TEDxCUSAT 2025.

## Features
- **Location-Based**: GPS tracking to unlock zones.
- **AR View**: Find 3D artifacts in the real world.
- **Story Integration**: Immersive unlocking mechanism.
- **Mock Mode**: Fully playable offline for development (`npm run dev`).

## Stack
- Next.js 14 + TypeScript
- TailwindCSS
- A-Frame / AR.js
- Supabase (Optional for Prod)

## Quick Start
```bash
npm install
npm run dev
```
